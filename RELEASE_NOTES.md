# v3.0.0 - 多语言 + 文件预览

## 新功能

### 多语言（i18n）
- 语言包系统：`lang/` 目录，内置中文（zh-CN）和英文（en）
- 所有 UI 文本统一通过 `t(key)` 函数管理
- 页面右上角语言切换下拉菜单，切换后即时生效并保持
- 支持 `Accept-Language` 请求头自动检测
- 支持自行添加更多语言文件

### 文件预览
- HMAC-SHA256 签名短时链接（10分钟有效），防止盗链
- 预览格式：文本、图片、PDF、视频、音频
- 视频/音频支持 Range 请求分段流式播放
- 管理页和上传查看页均集成预览按钮
- 不支持的格式自动隐藏预览按钮

### 安全增强
- 登录验证码（SVG 图形验证码）
- 文件预览签名机制，无法伪造

## 其他变更
- 全系统仅保留短链接（`/v/` 上传ID查看、`/f/` 文件下载）
- 存储目录从 `YYYY/MM/DD` 嵌套结构改为 `yyyyMMdd` 单层目录
- 后台管理页增加按批次删除功能
- 管理页路径列替代文件名列
- 移除 `formidable` 未使用依赖，清理调试日志

## 升级说明
- `config.json` 新增 `i18n` 配置项（自动添加默认值）
- 数据库结构不变，直接替换文件即可升级
- 首次启动会自动补齐历史数据的批次和短链字段
